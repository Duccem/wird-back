# Wird backend test

Este es el proyecto de backend hecho para la prueba tecnica de Wird

## Probar proyecto

### Con docker

Requisitos:

- docker

Comandos:

```bash
docker compose up
```

### Ambiente local

Requisitos:

- Node >= 20.X.X
- Redis

Comandos

```bash
npm install --save
npm run build
npm start
```

O

```bash
npm install --save
npm run dev
```
